# ITI Store

ITI Store is a simple Django mini project for managing books and authors.

The project demonstrates a **many-to-many relationship** between the two models, where:
- one book can have multiple authors
- one author can write multiple books

This project was built as part of my learning journey in Django and backend development during the **ITI Open Source Scholarship**.

## Features

- Add books
- Add authors
- Manage the relationship between books and authors
- Practice working with Django models
- Understand how to implement a many-to-many relationship

## Technologies Used

- Python
- Django
- MySQL

## Project Structure

The project includes two main models:

- **Book**
- **Author**

### Relationship
A **many-to-many relationship** is implemented between books and authors:
- A book can have multiple authors
- An author can write multiple books
